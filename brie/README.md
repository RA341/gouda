# brie

Frontend for gouda written in flutter.

## Pages

1. Book request History Page
    * Downloaded Details page
2. Mam Search Page
    * Book Details Page
3. Settings Page
    * Server Management (If admin)
        * Users
            * Credentials
            * Allowed Libraries
            * Notifications
        * Library Management
        * Mam Management Page
            * Mam token
            * Bonus Points
            * Cheese config
        * Torrent Client Settings
        * File Permissions
    * Individual User Management
        * Username/Pass
