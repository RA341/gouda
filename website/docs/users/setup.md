# Setup

Follow these steps to setup gouda

1. Configure Gouda for your [platform](installation).
2. Access the web interface at `http://localhost:9862` or whatever ip you are running gouda on
3. Navigate to the settings page and configure the following:
    - [MAM token](#myanonamouse-api-token)
    - File paths
    - Downloader settings

## MyAnonaMouse API Token

You'll need an active MyAnonaMouse account and an API token to use Gouda:

1. Log in to your MyAnonaMouse account
2. Head over to this [link](https://www.myanonamouse.net/preferences/index.php?view=security)
3. Generate an token
4. Use this token in Gouda config
